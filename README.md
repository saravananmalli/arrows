# Arrows (Frontend + Backend)

This is a minimal scaffold for a fullstack React (Vite) frontend and Node + Express backend using dummy data.

Figma prototype: https://www.figma.com/proto/rgHnPkysaRDA03RSaTBAqV/Arrows--Copy-?node-id=1-18359&t=a10nVL6UTz4r2vAT-0&scaling=min-zoom&content-scaling=fixed&page-id=0%3A1&starting-point-node-id=1%3A18359

Quick start

Run both services from the project root (recommended):

```bash
# from project root
npm install
npm run dev
```

Or run services individually:

Backend

```bash
cd backend
npm install
npm run dev
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Open the frontend dev server (Vite) and it will fetch dummy users from the backend at `http://localhost:4000/api/users`.
